## เริ่มเขียนโปรแกรมแรก

เปิด Editor และสร้างไฟล์ใหม่ ตั้งชื่อว่า `hello.c` และเขียนโค้ดข้างล่าง

```c
#include <stdio.h>

int main() {
  printf("Hello World!");
  return 0;
}
```

มาดูกันทีละส่วน

```c
#include <stdio.h>
```

`#include` เรียกว่า Preprocessor directive ใช้สำหรับเพิ่มเนื้อหาโค้ดจากไฟล์อื่น ในที่นี้ เราเรียกใช้ `stdio.h` ซึ่งเป็นไลบรารีสำหรับใช้งานฟังก์ชัน `printf()`

```c
int main() {
  ...
  return 0;
}
```

ส่วนนี้ เป็นการประกาศฟังก์ชัน ในที่นี้คือ ฟังก์ชันชื่อ `main()` ซึ่งเป็นฟังก์ชันหลักของการทำให้โปรแกรมทำงาน

```c
printf("Hello World!");
```

ส่วนนี้ เป็นการเรียกใช้ฟังก์ชัน `printf()` ซึ่งเป็นฟังก์ชันสำหรับแสดงผลข้อความออกทางหน้าจอโปรแกรม ในที่นี้เราจะแสดงผลข้อความ `Hello World!`

ต่อไป เราจะทำการคอมไพล์โค้ด ทำการเปิด Terminal ขึ้นมา และรันคำสั่ง

```bash
$ gcc hello.c
```

หลังจากรันคำสั่ง เราจะได้ไฟล์ `a.exe` ซึ่งเป็นโปรแกรมที่คอมไฟล์เรียบร้อยแล้ว
ทำการรันโปรแกรม โดยใช้คำสั่ง

```bash
$ a.exe
```

เราจะได้ผลลัพธ์โปรแกรมเป็นดังนี้

```plaintext
Hello World!
```

> ในฟังก์ชัน main จะเห็นว่าแต่ละบรรทัด หรือแต่ละคำสั่ง จะถูกคั่นด้วยเครื่องหมายอัฒภาค (Semicolon) `;` หากจบคำสั่งแต่ไม่ได้ใส่ `;` หลังจากการคอมไพล์โค้ด โปรแกรมจะส่งค่า error กลับมาให้ เช่น
>
> ```c
> #include <stdio.h>
>
> int main() {
>   printf("Hello World!")
>   return 0;
> }
> ```
>
> เมื่อคอมไพล์จะได้ผลลัพธ์เป็น
>
> ```plaintext
> hello.c: In function 'main':
> hello.c:4:25: error: expected ';' before 'return'
>    4 |   printf("Hello World!")
>      |                         ^
>      |                         ;
>    5 |   return 0;
>      |   ~~~~~~
>
> ```
>
> เมื่อแก้ไขตามที่คอมไพเลอร์บอก และคอมไพล์โค้ดอีกครั้ง โปรแกรมก็จะสามารถใช้งานได้

## การแสดงผลลัพธ์ในคอนโซล

จากหัวข้อที่ผ่านมา เราทราบว่าฟังก์ชัน `printf()` ที่อยู่ในไลบรารี `stdio.h` ใช้สำหรับแสดงผลข้อความ เราจะมาทำความรู้จักมากขึ้นในหัวข้อนี้

ในเบื้องต้น เมื่อเราจะแสดงผลข้อความด้วยฟังก์ชัน `printf()` เราจะต้องส่งค่าที่เป็นสตริง (String) เข้าไปโดยใช้เครื่องหมายคำพูดคู่ (Double quotes) ครอบข้อความที่เราต้องการแสดงผล
หากเราไม่ใส่ คอมไพเลอร์จะมองว่าค่านั้นเป็นตัวแปร และส่งผลให้เกิด error เมื่อคอมไพล์

```c
#include <stdio.h>

int main() {
  printf("Hello World 1!");
  printf("Hello World 2!");
  printf("Hello World 3!");
  return 0;
}
```

เมื่อเราลองรันโปรแกรมข้างต้น เราจะได้ผลลัพธ์เป็น

```plaintext
Hello World 1!Hello World 2!Hello World 3!
```

เนื่องจากฟังก์ชัน `printf()` ไม่มีการขึ้นบรรทัดใหม่ให้เอง เราจำเป็นจะต้องเพิ่มเข้าไปด้วยตัวเองโดยใช้อักขระพิเศษ `\
` ซึ่งเป็น Escape character ตัวหนึ่งที่ใช้สำหรับขึ้นบรรทัดใหม่

```c
#include <stdio.h>

int main() {
  printf("Hello World 1!\n");
  printf("Hello World 2!\n");
  printf("Hello World 3!");
  return 0;
}
```

ผลลัพธ์จะเป็นดังนี้

```plaintext
Hello World 1!
Hello World 2!
Hello World 3!
```

> เราสามารถเขียนรวมกันในหนึ่งฟังก์ชันได้ โดยยังได้ผลลัพธ์เดิม
>
> ```c
> int main() {
>   printf("Hello World 1!\nHello World 2!\nHello World 3!");
>   return 0;
> }
> ```

## เขียนคอมเมนต์ในภาษาซี

ในภาษาซี เราสามารถเขียนคอมเมนต์ได้สองแบบ ดังนี้

- Single-line : เริ่มต้นคอมเมนต์ด้วย `//` โดยสามารถเขียนได้ทั้งที่เป็นบรรทัดว่าง หรือบรรทัดที่มีโค้ดอยู่แล้วโดยต้องพิมพ์ต่อจากโค้ดได้

```c
#include <stdio.h>

int main() {
  // You can comment here
  printf("Hello World!"); // Or here
  return 0;
}
```

- Multi-line : เริ่มต้นคอมเมนต์ด้วย `/*` และเมื่อจบคอมเมนต์ต้องลงท้ายด้วย `*/`

```c
#include <stdio.h>

int main() {
  /* You can comment here 1
     You can comment here 2 */
  printf("Hello World!"); /* No Problem here */
  return 0;
}
```

## ตัวแปรในภาษาซี

ตัวแปร (Variable) ในภาษาโปรแกรมใช้สำหรับเก็บค่าข้อมูล ในภาษาซี ตัวแปรสามารถเป็นได้ทั้งตัวเลข อักษร หรือชนิดข้อมูลที่ซับซ้อนได้ ตัวอย่างเช่น

- `int` (Integer) : จำนวนเต็ม
- `float` (Float) : ทศนิยม
- `char` (Character) : อักขระเดี่ยว
- `char[]` (Array of char) : ชุดตัวแปรของอักขระเดี่ยว

ในการสร้างหรือนิยามตัวแปร เราต้องประกาศในรูปแบบ

```c
...
// [type] [varName] = [value];
int cost = 5;
...
```

จากตัวอย่างโค้ด เราประกาศตัวแปรชนิดจำนวนเต็มชื่อว่า `cost` โดยให้ค่าเริ่มต้นเป็น `5`
จะเห็นว่ามีการใช้เครื่องหมายเท่ากับ `=` ซึ่งในการประกาศตัวแปรใช้เป็นสัญลักษณ์ในการกำหนดค่า

เรายังสามารถประกาศตัวแปรโดยกำหนดค่าให้ทีหลังได้ ดังนี้

```c
...
/* [type] [varName];
   [varName] = [value] */
int cost;
cost = 5;
...
```

เราสามารถประกาศตัวแปรชนิดเดียวในบรรทัดเดียวได้

```c
...
int a = 1, b = 2, c = 3;

int x, y, z;
x = y = z = 4;
...
```

### Format Specifier

เราสามารถแสดงผลค่าตัวแปรโดยใช้ฟังก์ชัน `printf()` ได้ โดยใช้สิ่งที่เรียกว่า Format Specifier โดยสิ่งนี้จะทำหน้าที่เป็นตัวยึดตำแหน่ง (Placeholder) เพื่อนำค่าตัวแปรของเรามารวมกันกับข้อความได้ โดยเราจะต้องกำหนดชนิดของข้อมูลที่จะนำมาใส่ไว้ด้วย ตัวอย่างเช่น

```c
...
int cost = 5;
printf("%d", cost);
...
```

ผลลัพธ์จะเป็น

```plaintext
5
```

```c
...
int cost = 5;
char currency = '$'
printf("This apple is %d%c", cost, currency);
...
```

ผลลัพธ์จะเป็น

```plaintext
This apple is 5$
```

> `%d` ใช้แสดงผล `int` > `%f` ใช้แสดงผล `float` > `%c` ใช้แสดงผล `char`

### เปลี่ยนค่าตัวแปร

เมื่อเรากำหนดค่าให้กับตัวแปรอีกครั้ง ค่าเดิมจะถูกแทนที่ด้วยค่าใหม่

```c
...
int cost = 5;
printf("%d", cost);
cost = 6;
printf("%d", cost);
...
```

```plaintext
5
6
```

จะเห็นว่าการแสดงผลครั้งแรกผลลัพธ์เป็น `5` แต่เมื่อเปลี่ยนค่าใหม่ แล้วแสดงผลอีกครั้ง ผลลัพธ์จะได้เป็น `6` และหลังจากคำสั่งนี้จนจบโปรแกรมค่าของ `cost` ก็จะเป็น `6` จนกว่าจะเปลี่ยน

### การตั้งชื่อตัวแปร

ตัวแปรทุกตัวต้องมีชื่อ และชื่อต้องไม่ซ้ำกันในขอบเขต (Scope) สามารถตั้งชื่อใดก็ได้แต่ต้องอยู่ในกฏการตั้งชื่อ และต้องไม่เป็นคำสงวน (Reserved word) ที่ใช้ในภาษาซี

- ต้องประกอบด้วยตัวอักษร (ภาษาอังกฤษ) ตัวเลข และเครื่องหมาย `_` (Underscore) เท่านั้น
- ต้องขึ้นต้นด้วยตัวอักษร หรือ `_`
- ตัวแปรในภาษาซีมีลักษณะ Case-sentitive ตัวพิมพ์เล็กจะเป็นคนละตัวกับตัวพิมพ์ใหญ่

> ในทางปฏิบัติ ควรตั้งชื่อตัวแปรให้สื่อความหมาย ความยาวคำไม่สั้นและไม่ยาวเกินไป
